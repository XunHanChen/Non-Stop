URP Setup: 

Support Unity versions
2019.4.0 or higher

First delete the Builtin folder and unpack the URP-Wet shaders Puddles.unitypackage. In order to use the transparent shader with the Universal Render Pipeline
you will need to enable the Opaque Texture toggle in the pipeline asset inspector(for more information check the documentation linked below).



Documentation link : https://docs.google.com/document/d/1SDgEmudN0OSQt-ARpWna50QLQxZ3idX1p2DWUJ-4Ius/edit?usp=sharing

Setp-by-Step Tutorial 
_____________________
Videos
- Wet Shader Flooded Ground - Tutorial : https://youtu.be/Qfv_gW7AKig
- Wet Shader Rainy Puddles - Tutorial    : https://youtu.be/F9U3KB3C8tE


Create a rainy puddle material :
- Right click within a folder in the project window.
- Create/Material
- Select the new material created
- In the inspector window, go to the shader drop-down list
- Select CiconiaStudio/CS_Wet Animation/...
Done

For more information on shader properties, please refer to the documentation linked above.
